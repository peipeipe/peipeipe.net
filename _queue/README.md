# 日記投稿キュー

このディレクトリには、Webフォームから投稿された日記エントリが一時的に保存されます。

## 仕組み

1. Webフォームから投稿されると、JSONファイルがここに作成されます
2. GitHub ActionsがJSONファイルの変更を検知します
3. 自動的に日記エントリに変換して`_diary/`に保存されます
4. 処理完了後、JSONファイルは削除されます

## ファイル形式

```json
{
  "content": "投稿内容",
  "username": "投稿者名",
  "timestamp": "2025-06-24T15:30:00Z"
}
```
